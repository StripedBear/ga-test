<p><em>Some great programming books recommended by our former team member Aleksandr Novikov which are perfect for those short on inspiration and time. </em></p>
<h6 id="h-grokking-algorithms-by-aditya-y-bhargava"><a href="https://www.adit.io/posts/2016-05-25-Grokking-Algorithms-Is-Out.html">Grokking Algorithms</a> <em>by Aditya Y. Bhargava </em></h6>
<figure class="wp-block-image size-large"><img alt="Grokking Algorithms by Aditya Y. Bhargava review" class="wp-image-3168" height="656" loading="lazy" sizes="(max-width: 1024px) 100vw, 1024px" src="Untitled-design-3.jpg" srcset="https://generalarcade.com/content/uploads/2025/01/Untitled-design-3.jpg 1024w, https://generalarcade.com/content/uploads/2025/01/Untitled-design-3-300x192.jpg 300w, https://generalarcade.com/content/uploads/2025/01/Untitled-design-3-768x492.jpg 768w" width="1024"/></figure>
<p>This was one of my first programming books. It’s an easy and fun read to grasp basic data structures and algorithms.</p>
<h6 id="h-code-complete-by-steve-mcconnell"><a href="https://stevemcconnell.com/books/">Code Complete</a> <em>by Steve McConnell </em></h6>
<figure class="wp-block-image size-large"><img alt="Code Complete by Steve McConnell review" class="wp-image-3171" height="656" loading="lazy" sizes="(max-width: 1024px) 100vw, 1024px" src="teamsbooksgeneralarcade-1.jpg" srcset="https://generalarcade.com/content/uploads/2024/07/teamsbooksgeneralarcade-1.jpg 1024w, https://generalarcade.com/content/uploads/2024/07/teamsbooksgeneralarcade-1-300x192.jpg 300w, https://generalarcade.com/content/uploads/2024/07/teamsbooksgeneralarcade-1-768x492.jpg 768w" width="1024"/></figure>
<p>A classic that demonstrates good coding practices, no matter the language. I’ve re-read it multiple times (though selectively).</p>
<h6 id="h-game-programming-patterns-by-robert-nystrom"><a href="https://gameprogrammingpatterns.com/">Game Programming Patterns</a><em> by Robert Nystrom </em></h6>
<figure class="wp-block-image size-large"><img alt="Game Programming Patterns by Robert Nystrom review" class="wp-image-3173" height="656" loading="lazy" sizes="(max-width: 1024px) 100vw, 1024px" src="teamsbooksgeneralarcade-2.jpg" srcset="https://generalarcade.com/content/uploads/2024/07/teamsbooksgeneralarcade-2.jpg 1024w, https://generalarcade.com/content/uploads/2024/07/teamsbooksgeneralarcade-2-300x192.jpg 300w, https://generalarcade.com/content/uploads/2024/07/teamsbooksgeneralarcade-2-768x492.jpg 768w" width="1024"/></figure>
<p>This book illustrates the application of programming patterns specifically in game development. It’s written in simple language and is quite easy to follow.</p>
<h6 id="h-game-engine-architecture-by-jason-gregory"><a href="https://www.gameenginebook.com/">Game Engine Architecture</a><em> by Jason Gregory </em></h6>
<figure class="wp-block-image size-large"><img alt="Game Engine Architecture by Jason Gregory review" class="wp-image-3175" height="656" loading="lazy" sizes="(max-width: 1024px) 100vw, 1024px" src="teamsbooksgeneralarcade-3.jpg" srcset="https://generalarcade.com/content/uploads/2024/07/teamsbooksgeneralarcade-3.jpg 1024w, https://generalarcade.com/content/uploads/2024/07/teamsbooksgeneralarcade-3-300x192.jpg 300w, https://generalarcade.com/content/uploads/2024/07/teamsbooksgeneralarcade-3-768x492.jpg 768w" width="1024"/></figure>
<p>A comprehensive guide packed with knowledge about systems used in game engines.</p>
<h6 id="h-the-clean-coder-a-code-of-conduct-for-professional-programmers-by-robert-c-martin"><a href="http://cleancoder.com/files/cleanCodeCourse.md">The Clean Coder: A Code of Conduct for Professional Programmers</a><em> by Robert C. Martin </em></h6>
<figure class="wp-block-image size-large"><img alt="The Clean Coder: A Code of Conduct for Professional Programmers by Robert C. Martin review" class="wp-image-3177" height="656" loading="lazy" sizes="(max-width: 1024px) 100vw, 1024px" src="teamsbooksgeneralarcade-4.jpg" srcset="https://generalarcade.com/content/uploads/2024/07/teamsbooksgeneralarcade-4.jpg 1024w, https://generalarcade.com/content/uploads/2024/07/teamsbooksgeneralarcade-4-300x192.jpg 300w, https://generalarcade.com/content/uploads/2024/07/teamsbooksgeneralarcade-4-768x492.jpg 768w" width="1024"/></figure>
<p>A great read for developing soft skills for developers, authored by the renowned Uncle Bob.</p>
